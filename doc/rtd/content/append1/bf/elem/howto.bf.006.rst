This module demonstrates how to use classes Buffer and BufferElement. An example output below shows the functionality of buffer element and buffer in MLPro. This implementation shows, how to set buffer base, maximum buffer size, buffer elements, add data and clear the buffer. To get the results as shown below, please run the script provided in the example code section of this document.

Game Boards

Bulk Goods Plant

Ver. 1.0.4 (2021-11-29)

This module provides an environment of Bulk Good Laboratory Plant (BGLP) following GT interface. This module provides game board classed based on BGLP environment of the reinforcement learning pool.

class mlpro.gt.pool.boards.bglp.BGLP_GT(p_logging=True, t_step=0.5, t_set=10.0, demand=0.1, lr_margin=1.0, lr_demand=4.0, lr_energy=0.001, margin_p=[0.2, 0.8, 4], prod_target=10000, prod_scenario='continuous')

Bases: mlpro.rl.pool.envs.bglp.BGLP, mlpro.gt.models.GameBoard

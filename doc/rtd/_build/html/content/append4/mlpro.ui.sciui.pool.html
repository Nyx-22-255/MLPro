Ver. 1.0.1 (2021-07-07)

This module provides the reusable SciUI component ‘Interactive Input Space’. It represents a 2D or 3D input space based on a mathematical space and allows manual (2D) and automated (2D,3D) input creation. It consists of time depending subplots for each input dimension and a 2D/3D main subplot for the input space. To be reused and enriched in own SciUI scenarios.

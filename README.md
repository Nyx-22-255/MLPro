# MLPro - Machine Learning Professional
Machine Learning Professional - A Synoptic Framework for Standardized Machine Learning Tasks in Python

MLPro provides complete, standardized, and reusable functionalities to support your scientific research, educational tasks or industrial projects in machine learning.

Project MLPro was started in 2021 by [Automation Technology and Learning Systems team at Fachhochschule SÃ¼dwestfalen](https://www.fh-swf.de/de/forschung___transfer_4/labore_3/labs/labor_fuer_automatisierungstechnik__soest_1/standardseite_57.php)


# Key Features
-   Reinforcement Learning with Single-Agents, Multi-Agents (MARL) or model-based Agents (MBRL)
-   Cooperative Game Theory
-   Incorporation of established ML packages like OpenAI Gym, PettingZoo, Stable Baselines 3, Hyperopt
-   Numerous executable self study examples
-   Usability in scientific, industrial and educational contexts
-   Extensible, maintainable, understandable
-   Attractive UI support (available soon)


# Development
-   Test-driven development in GitHub (CI/CD concept)
-   Consequent Object-Oriented Programming
-   Clean code
